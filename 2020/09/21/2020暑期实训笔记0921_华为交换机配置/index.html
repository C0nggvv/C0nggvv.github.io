eNSP配置

交换机基本配置

快捷键

按?能获得帮助信息，按TAB键可自动补全

配置视图

<Huawei>         //用户模式 
<Huawei> system-view //切进系统视图  
[Huawei]         //系统视图 
[Huawei] interface GigabitEthernet 0/0/1 //切进接口视图 
[Huawei-GigabitEthernet0/0/1] //接口视图

退回前一个视图：quit

快速返回用户视图：return 或 Ctrl+Z

查看版本：display version

配置设备名：sysname name1

查看当前配置：display current-configuration

保存当前配置：save

启用接口：shutdown

关闭接口：undo shutdown

关闭终端显示功能：undo terminal monitor

设置终端密码

[lsw1] user-interface console 0 
[lsw1-ui-console0]authentication-mode password   //启用密码认证 
[lsw1-ui-console0] set authentication password cipher Taren1  
//设置终端登录密码为 Taren1

为vlan1接口配置IP地址

/**
 * 交换机的接口用来连计算机网卡，默认不能配置IP地址
 * 为方便管理，需要使用虚接口vlan1
 */
[Switch1]interface vlan1 //进虚接口vlan1配置 
[Switch1-Vlanif1]ip address 192.168.1.100 24 //设置IP地址

远程管理交换机

[Switch1]user-interface vty 0 4  //进远程登录配置视图 
[Switch -ui-vty0-4] user privilege level 3  //设置用户登录 
[Switch1-ui-vty0-4]authentication-mode aaa  //启用AAA认证
[Switch1] aaa  //进AAA配置视图 
[Switch1-aaa] local-user ntdadm password cipher Taren1  //添加用户ntdadm 
[Switch1-aaa] local-user ntdadm service-type telnet //选择控制方式telnet

远程连接：telnet IP地址

路由器基本配置

配置视图

<Huawei>         //用户视图 
<Huawei>system-view   
[Huawei]         //系统视图 
[Huawei]interface GigabitEthernet 0/0/0 
[Huawei-GigabitEthernet0/0/0]      //接口视图

配置路由器主机名：sysname name1

配置终端密码

<Router> system-view 
[Router-ui-console0]authentication-mode password
输入密码内容  //配置密码

配置接口IP地址：ip address IP地址 掩码位数

配置静态路由：ip route-static 目的网段 子网掩码 下一条IP地址

查看路由表：display ip routing-table

远程管理路由器

设备管理AAA身份认证模式
[Router1] user-interface vty 0 4 //进入远程登录配置视图 
[Router1 -ui-vty0-4] user privilege level 3        //设置用户登录 
[Router1 -ui-vty0-4] authentication-mode aaa     //启用AAA认证
/**
 * 配置AAA认证细节
 * 添加用户ntdadm，密码设为 Taren1
 * 选择控制方式 telnet 
 */
[Router1] aaa      //进AAA配置视图 
[Router1 -aaa] local-user ntdadm password cipher Taren1 //添加用户ntdadm 
[Router1 -aaa] local-user ntdadm service-type telnet     //选择控制方式telnet

远程登录路由器：telnet IP地址

防火墙终端命令配置

查看接口：display ip interface brief

配置接口

interface GigabitEthernet1/0/1              //选择接口
  ip address 192.168.1.254 255.255.255.0      //配置IP地址
  service-manage ping permit                 //允许Ping服务
interface GigabitEthernet1/0/2  
  ip address 200.1.1.254 255.255.255.0 
firewall zone trust                          //添加trust接口
  add interface GigabitEthernet1/0/1 
firewall zone untrust                          //添加untrust接口
  add interface GigabitEthernet1/0/2

配置安全策略

security-policy  
  rule name out    
    source-zone trust    
    destination-zone untrust    
    action permit

配置默认路由：ip route-static 目的IP 子网掩码 下一条IP

eNSP卸载の正确方法

  1. 先在应用-卸载中卸载eNSP、Oracle VM VirtualBox、WinPcap、wireshark四个软件
  2. 删除目录，一般在C:\Program Files\Huawei
