2.YUM本地安装源配置

创建软件仓库的配置文件

mkdir /root/yum.repo                    #创建备份目录
mv /etc/yum.repos.d/* /root/yum.repo    #移动备份文件
vi /etc/yum.repos.d/local.repo            #创建新的配置文件

编写repo文件并指向光盘镜像文件挂载点

在该新的配置文件local.repo内写入如下内容

[local]
name=local
baseurl=file:///mnt/cdrom        #指向镜像文件的挂载目录
enabled=1                        #1表示可用状态
gpgcheck=0                        #不检验待安装RPM包

清除缓存

yum clean all            #清除缓存
yum makecahe            #把YUM源缓存到本地

安装GNOME桌面

yum grouplist                         #显示所有可用的的套件，其中有’GNOME Desktop'
yum groupinstall 'GNOME Desktop'    #安装GNOME桌面，名字与上面一步出现的名字一致

重启&启动桌面

reboot                    #重启
startx                    #启动桌面服务
