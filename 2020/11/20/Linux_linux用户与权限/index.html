linux用户与权限

未完待更…

  • 切换用户 su [-] [用户名]
    • - 表示切换时是否带切换用户的环境变量
    • 如果不带用户名则是指普通用户切换到root用户
      • 切换到root用户 su - 或 su - root
      • 切换到admin用户 su - admin
  • 查看当前系统各用户的账户信息 cat /etc/passwd
    • 使用grep确认系统中是否已有hx 用户 grep hx /etc/passed
  • 创建新用户 useradd [参数] 用户名
    • 创建hx 新用户 useradd hx
  • 设置用户账号口令（密码） passwd [参数] 用户名
    • 若不使用参数及不指定用户名，则为当前用户修改自身口令
      • 设置hx 的密码passwd hx
  • 创建新的用户组 groupadd [-r] 组名
    • -r 带r参数表示建立系统组，否则建立普通组
      • 建立 xinan 用户组 group xinan
  • 在用户组中添加/删除指定用户 gpasswd [参数] 用户名 组名
    • gpasswd -a username groupname 将指定用户添加到用户组中
    • gpasswd -d username groupname 将指定用户从用户组中删除
    • gpasswd -A username groupname 设指定用户为组的管理员
      • 将hx假如xinan组中 gpasswd -a hx xinan
  • 修改文件的属主与属组 chown [选项] [所有者][:[组]] 文件列表
    • 修改文件f1的属组为wdg chown :wdg f1
  • 修改文件权限 chmod [选项] 模式[,模式] 文件名 或 chmod 八进制模式 文件名
    • 字母模式修改权限
      • 用户对象
        • u user用户对象
        • g group同组用户
        • o others其他用户
        • a all以上所有用户
      • 操作符号
        • + 添加某个权限
        • -取消某个权限
        • =赋予给定权限并取消其他所有权限
      • 操作权限
        • r 可读
        • w 可写
        • x 可执行
      • 案例
        • chmod u+x,g+w,o-r f1
        • chmod a=rw f2
    • 数字形式修改权限
      • 案例 chmod 644 f1
  • 用exit退出用户命令，返回root命令 exit
