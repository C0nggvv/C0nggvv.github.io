linux基本命令

  • 查看当前所在目录 pwd
  • 列出此目录下的文件和目录 ls
    • -a 列出此目录下包括隐藏文件在内的所有文件和目录 [隐藏文件特点：文件名第一个字符为 .]
    • -l 列出详细信息(长格式) [目录文件第一个字母为 d ,普通文件第一个字母为 - ]
    • -i 显示inode节点号
  • 切换文件目录
    • 到/etc目录 cd /etc
    • 进入此目录下的tmp目录 cd tmp(开头无 / )
    • 上一级目录 cd ..
    • 当前目录 cd .
    • 用户主目录 cd
  • 查看文件内容
    • cat查看文件file1内容 cat file1
      • 显示行号 cat -n file1
    • less查看文件内容 less file1 [按q退出]
    • more分屏显示文件内容 more file1
  • 用man命令查看ls命令的使用手册 man ls
  • date查看系统时间 date
  • 切换用户su [-][用户名]
    • - 切换时同时切换用户的环境变量
    • 若不带用户名则表示切换到root用户 su
  • 文件检索命令grep 字符串 [文件]
    • grep命令在profile文件中对关键字then进行查询 grep then profile
  • 查找文件命令 find [路径][表达式]
    • find / -name test从根目录/查找名为test的文件
      • 找root用户自己的主目录下的所有名为newfile的文件 find ~ -name newfile
  • 输出重定向
    • 直接写入（覆盖） >
    • 追加（到文件末尾）>>
      • ls命令在终端上显示的当前目录中的文件列表重定向到文件list中 ls > list[先创建文件再添加内容]
      • 把ls命令在终端上显示的当前目录中的文件列表重定向到文件list中（追加方式）ls >> list
  • 创建测试目录test mkdir test
    • -p 递归生成目录：mkdir -p dir/subdir
  • 创建一个新的空文件newfile
    • 利用touch命令 touch newfile
  • cp复制命令
    • 复制test目录为testbak目录作为备份 cp -r test testbak
      • -r 该目录及其子目录递归复制
    • 复制某目录下（/etc/profile）的文件到当前目录(./ )cp /etc/profile ./[最后/可加可不加]
    • 复制文件（profile)为新文件(profile.bak) cp profile profile.bak
    • 将file1复制到目录dir1中，复制后的文件名为file1.bak cp file1 dir1/file1.bak
  • mv移动命令
    • 把文件file1移动到test目录下 mv file1 test
    • 修改文件file1的文件名为file2 mv file1 file2
  • rm删除命令
    • 删除文件file1 rm file1
    • 删除目录dir rm -r dir
      • -r 递归删除目录文件
      • -f 不发出提示确认
  • rmdir 删除空目录命令 rmdir [-p][目录名]
    • -p 递归删除，若子目录删除后父目录为空则递归删除父目录
  • 给文件创建连接ln [-s] 源文件 [目标文件]
    • 给文件profile创建一个符号链接lnsprofile ln -s profile lnsprofile
    • 给文件profile创建一个硬链接lnhprofile ln profile lnhprofile
